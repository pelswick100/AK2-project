# AK2-project
 Kalkulator zmiennoprzecinkowy 64-bitowy napisany za pomocą biblitoeki QT w języku Assmebler oraz C++
 Kalkulator będzie wykonywał następujące działania:
  - dodawanie
  - odejmowanie
  - mnożenie
  - dzielenie
  - pierwiastek kwadratowy
  - potęgowanie
  - funkcja logarytmiczna (log)
  - funkcja sinus
  - funkcja cosinus
  - funkcja tangens
  - funkcja cotangens
  - NWD
  - NWW
  - ln
